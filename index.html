Sobre o Projeto Anna Bella

Sobre este projeto de estudo, o primeiro que eu fiz junto com o instrutor Jamilton Dasmasceno pela Udemy no curso de Desenvolvimento Web, sobre orientações com apenas html e css

Este foi o ponto de partida para a minha paixão que é hoje desenvolver sites e ver como tudo é possível, basta estudar, práticar e ter dedicação.

Este Blog é bem simples do ponto de vista de programação, mas decidi coloca-lo em meus portifólios justamente porque foi o ponto de partida, onde tudo começou.....Claro que eu tive que fazer ajustes ao menos para ele ficar responsivo, já que o mesmo era estático, não quis mudar muita coisa do projeto inicial feito no curso, para não perder a essência.
